# Spine

Spine is a configuration-first ingestion framework for teams tired of rebuilding the same auth, retry, pagination, backfill, and loading logic for every new source, table, and destination. It is a modular runtime with one execution model, one config shape, and clear extension points for services, parsers, collectors, and loaders.
